# AXIOM Terminal Feature Roadmap

## v1.0 Current

- Modular PowerShell profile
- Windows Terminal visual snippet
- AXIOM CRT background asset
- Terminal editor integration
- Safe verification/audit wrappers
- Read-only SQLite inspection
- Git status/diff inspection
- Extension module template
- Read-only dashboard, readiness, manifest, model, budget, event, queue, and task panels
- Docs/navigation panel
- Runtime/tool/UI boundary maps
- Phase 5 manual/test-only agent visibility

## Candidate future additions

Add only when AXIOM runtime prerequisites exist and the safety boundary is ratified.

1. Calibration-status dashboard.
2. ContextBuilder/TokenEstimator inspection commands.
3. Handoff diff viewer.
4. Operator-critical summary panel that reduces duplicate dashboard/readiness/report output.
5. Gateway posture panel showing model/network/sandbox/memory boundary state.

Do not add automatic scheduler-to-executor or scheduler-to-agent integration helpers until explicitly approved.
